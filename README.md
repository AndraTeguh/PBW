## ðŸ“° Deskripsi Aplikasi

NewsApp adalah aplikasi sistem manajemen berita sederhana yang memiliki fitur utama seperti :

-   Menampilkan daftar berita & detail lengkap
-   Menampilkan dan mengatur informasi wartawan
-   Menampilkan dan mengatur komentar pada setiap berita
-   Admin Panel berbasis Filament

Fitur yang Ditambahkan:

-   **Mengelola data wartawan sebagai penulis berita**
-   **Admin panel dengan Filament untuk manajemen data (CRUD)**
-   **Melihat daftar komentar dan dapat menghapus komentar**
